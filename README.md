# XTouchTouchPad

X (Xorg) Touch (Touch-as-click) TouchPad

Enable TouchPad touch-as-click for Xorg

## Usage

```shell
$ git clone https://github.com/Trumeet/X-Touch-Touchpad.git
$ makepkg -ci
```
